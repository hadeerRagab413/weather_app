Weather App 🌤️

Weather App is a simple web application that shows the current weather for any city using a Weather API.

Features ✨
-Display current weather for any city.
-Weather details include:
 Temperature
 Weather condition (Sunny, Cloudy, Rain, etc.)
-Simple and user-friendly interface.

Installation 💻
Clone the repository:
git clone https://github.com/hadeerRagab413/weather_app.git
Go to the project folder:
cd weather-app
Install dependencies:
npm install
Start the project:
npm run start
Open http://localhost:3000
 in your browser.
 
API Used 🌐
This project uses OpenWeatherMap API (or your chosen weather API).
You need to get an API Key and add it to the project:
REACT_APP_API_KEY=28cd7afcfbc8c160ed654d73cf0601df

Technologies 🛠️
React.js
JavaScript
CSS / SASS
Axios (for API requests)

How to Use 🏃‍♀️
Open the app.
Type the city name you want to check.
Click Search.
The current weather and details will be displayed.
